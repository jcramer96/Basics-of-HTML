What is HTML?

HTML is short for "HyperText Markup Language." All that means is that it's a language for describing webpages using ordinary text. Every website you've ever visited is an HTML file, or a series of HTML files linked together.

Introduction to HTML Tags

HTML Tags are the keywords that define how a web page must display and structure its content. Most tags will have an opening and closing tag. An opening tag is a keyword displayed inside of <>brackets (i.e. <keyword>), and a closing tag is the same tag but with a forward slash in the front (i.e. </keyword>.

Note: There are some tags that are self-closing, meaning that the addition of a closing tag is not necessary.

Foundational HTML Tags

These tags appear in every HTML document and are the core building blocks of all webpages:

<html>
This is a container tag that holds all other elements within the HTML.
<head>
This tag contains the HTML's metadata and defines the document title, character style, and scripts.
<title>
This tag defines the title displayed on the browser's title bar.
<body>
This is a container tag that stores all the content within and HTML document.
Basic HTML Structure

Now that we know the foundational HTML tags, the basic outline of an HTML webpage is as follows:

<html>
<head>
<title> Website Title </title>
</head>
<body>
Website Content
</body>
</html>

Text Formatting Tags

These tags help structure and organize text so that it is easier to read, and helps emphasize categories, section, or keywords.

<h!>
This is a header tag that helps section off and organize text. The "!" should be changed to numbers 1 through 6 to determine how large or small the header should be (1=largest, 6=smallest); e.g. "h3".
<p>
This tag defines a paragraph of text.
<b>
This tag bolds the text.
<i>
This tag italicizes the text.
<u> or <em>
This tag underlines the text.
<blockquote>
This tag places a text block quote.
Section Division Tags

These tags are additional tools to help organize the text into sections and divisions.

<div>
This is a block-level container tag that defines a division or section
<span>
This is an inline container used to mark up the text within a document.
<br>
This tag inserts a line break.
<hr>
This tag inserts a horizontal line.

List Tags

These tags are used to create lists of items that stand out and are easy to read.

<ol>
This is an ordered list tag and it's used to list items in a specific sequence.
  • <ol type="A">
  • <ol type="a">
  • <ol type="1">
  • <ol type="i">
  • <ol type="I">
<ul>
This is an unordered list tag and it's used to list items in no particular sequence. Are automatically applied as bullet points, but can be further specified with the following "type" attributes:
  • <ul type="disc">
  • <ul type="circle">
  • <ul type="square">
<li>
These are the listed items in either an ordered or unordered list.
<dl>
This is a container tag for a definition list.
<dt>
This tag contains the term or phrase being defined.
<dd>
This tag contains a definition for a term or phrase.

Form Tags

These tags insert a form for user input.

<form>
This tag groups various inputs into a form and determines where the information will go.
<input>
This tag is interactable and comes in many different forms:
  • Text
  • Password
  • Checkbox
  • Submit
  • Name
  • Value
<select>
This tag creates a dropdown menu for item selection.
<textarea>
This tag creates a box for the user to write comments or responses.

Miscellaneous Tags

Here is a list of other frequently used HTML tags

<img src="url" alt="text">
This tag inserts an image into the HTML document. The "src" attribute retrieves the image from a URL to display. The "alt" attribute will display alternative text of your choosing if the image fails to load.
<a href="url">
This tag serves as a basic link to another webpage as indicated by the "href" attribute.
<!--...-->
This tag is used to make comments within the HTML source code.
<meta>
This tag defines the information about the data within an HTML document. It is used by internet browsers to determine how to display content and by search engines to find specific keywords.
